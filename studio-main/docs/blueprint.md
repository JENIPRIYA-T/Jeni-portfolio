# **App Name**: PortfoliAI

## Core Features:

- Profile Display: Displays personal information, career objectives, and contact details.
- Skills Showcase: Categorized presentation of technical skills, including programming languages, tools, and hardware/IoT.
- Project Portfolio: Detailed showcase of completed projects with descriptions, technologies used, and roles.
- Certification Display: List of acquired certifications relevant to the field of Computer Science and Engineering.
- Internship Experience: Details of internship experiences, focusing on responsibilities and technologies used.
- Resume Download: Allows visitors to download a PDF version of the resume.
- AI-Powered Content Suggestions: AI tool suggests improvements for the project descriptions, leveraging Oracle Cloud Infrastructure 2024 generative AI to ensure professional quality.

## Style Guidelines:

- Primary color: Subtle blue (#79A3B1) for a professional and calming feel.
- Background color: Light grey (#E9E9E9) to ensure readability and a clean look.
- Accent color: Teal (#398D93) for interactive elements and highlights, providing a subtle contrast.
- Body text: 'Inter' sans-serif font to ensure great readability.
- Headlines: 'Space Grotesk' sans-serif for a modern, clear heading style.
- Flat, line-based icons for sections to ensure visual clarity and a modern aesthetic.
- Clean, responsive layout with a one-page design for easy navigation.
- Subtle hover effects on project cards and buttons to enhance user interaction.
